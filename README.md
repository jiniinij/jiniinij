# Hi there, I'm Sojin Park! 👋

### Interaction Designer & Creative Technologist

I design and implement interactive systems that expand web-based interactions into physical spaces. Beyond mere visual representation, I focus on building **sustainable code architectures** and **clear UX flows** in real-world environments. My goal as a developer is to build environments that satisfy both technical stability and sensory experience, bridging the gap between artistic intuition and engineering.
